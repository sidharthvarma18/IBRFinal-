## 📊 Dashboard Sections
1. **Spending Trends** — Line charts, YoY heatmap, area chart
2. **Descriptive Statistics** — Tables, box plots, skewness/kurtosis
3. **Correlation Analysis** — Heatmap, scatter plots, r-values
4. **Regression Analysis** — OLS results, interactive scatter with trendlines
5. **India vs UAE** — Radar chart, t-test, fee ratio comparison
6. **Bank Deep-Dive** — Per-bank financial metrics explorer
7. **Global Benchmarking** — Multi-market radar, lounge unit economics, policy lessons

---

## 🗃️ Dataset
- **10 banks**: HDFC, ICICI, SBI Card, Axis, Kotak (India) + Emirates NBD, FAB, ADCB, Mashreq, DIB (UAE)
- **Period**: 2016–2025 (10 years)
- **Variables**: Annual Spends, Cards in Force, Receivables, Interest Rate, Fee & Commission Income
